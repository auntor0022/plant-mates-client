#  Project Name - Plant Mates 🌿

## 🔗 Live Site URL
[https://plant-mates-app.web.app/](https://plant-mates-app.web.app/)

## 🌟 Features
- 🔐 **Authentication System**: Secure email/password login and Google sign-in powered by Firebase.
- 🌱 **CRUD Garden Tips**: Authenticated users can add, edit, and delete gardening tips with public/private visibility.
- 👨‍🌾 **Explore Gardeners**: Browse local gardener profiles with experience, status, and total shared tips.
- 🪴 **Top Trending Tips**: View the most liked gardening tips on the home page.
- 🌗 **Dark/Light Theme Toggle**: Switch between light and dark modes from the navbar.
- 📊 **Tip Filtering System**: Filter tips based on difficulty level (Easy, Medium, Hard).
- ❤️ **Like System**: Users can like tips, and total likes are updated in real time.
- 🚀 **Fully Responsive**: Optimized for mobile, tablet, and desktop viewing.
- 🛠️ **Integrated Animations**: Uses Lottie & React Awesome Reveal for smooth UI transitions.

## 📝 Project Description
**GreenThumb** is a dynamic web application built for gardening lovers. It offers a community-driven platform to share personal gardening experiences, learn from other enthusiasts, find local gardeners, and post or attend gardening events. With a sleek and responsive design, real-time data handling, and interactive UI features, GreenThumb brings together both novice and seasoned gardeners into one helpful hub.
